## Hardware Requirements

### Minimum Requirements
* NVIDIA GPU with 8GB VRAM
* 16GB System RAM
* CUDA 11.7 or newer

### Recommended Setup
* NVIDIA GPU with 24GB+ VRAM
* 32GB+ System RAM
* NVMe SSD for model storage
